# [到达目的地的方案数](https://leetcode.cn/problems/number-of-ways-to-arrive-at-destination/)

=== "迪杰斯特拉"

    使用迪杰斯特拉遍历时，如果再一次遍历到了点 $u$，有什么特殊的地方呢？如果点 $u$ 在此之前已经遍历，说明点 $u$ 的最短路径一定是确定下来了的，如果在一次遍历到了该点，该次遍历带来的路径长度一定不会比最短路径还短，要么等，要么大；

    如果等，就说明又有一条新的最短路径抵达了该点，可以记录下来。

    

    ---

    写法一：

    如果相等的时候也允许放入队列中，此时虽然可以更新后续的子节点，但是空间复杂度和时间复杂度很有可能达到 $m ^ 2$ 级别.

    因为有很多冗余项.
    
    空间复杂度很高！

    ```java
    class Solution {
        long inf = ~(1L << 63);
        int N;
        long mod = 1000000007L;

        class Node {
            int a; long b; long c;
            Node(int x, long y, long z) {
                a = x; b = y; c = z;
            }
        }

        int n;
        int[][] es;
        int[] g, e, ne;
        long[] d;
        long[] md;
        long[] ct;
        long[] on;
        int cur;

        void add(int a, int b, int c) {
            e[cur] = b; d[cur] = c;
            ne[cur] = g[a]; g[a] = cur;
            cur ++;
        }

        java.util.Queue<Node> qu;


        public int countPaths(int ns, int[][] roads) {
            n = ns;
            es = roads;

            N = n * n;

            g = new int[N];
            e = new int[N];
            ne = new int[N];
            d = new long[N];
            md = new long[N];
            ct = new long[N];
            on = new long[N];

            for (int i = 0; i < N; i ++) {
                g[i] = -1; e[i] = -1; ne[i] = -1;
                md[i] = inf;
            }

            for (int[] nd : es) {
                add(nd[0], nd[1], nd[2]);
                add(nd[1], nd[0], nd[2]);

                // System.out.println(nd[0] + " " + nd[1] + " " + nd[2]);
            }

            qu = new PriorityQueue<>(new Comparator<Node>(){
                public int compare(Node x, Node y) {
                    if (x.b < y.b) return -1;
                    if (x.b > y.b) return 1;
                    return 0;
                }
            });

            qu.offer(new Node(0, 0, 1));

            while (qu.size() > 0) {
                Node aa = qu.poll();

                // System.out.println(aa.a + ": " + md[aa.a] + " -- " + aa.b + " -- " + aa.c);

                if (on[aa.a] > 0 && md[aa.a] < aa.b) continue;

                on[aa.a] = 1;
                md[aa.a] = aa.b;
                ct[aa.a] += aa.c;
                ct[aa.a] %= mod;

                for (int i = g[aa.a]; i > -1; i = ne[i]) {
                    qu.offer(new Node(e[i], aa.b + d[i], aa.c));
                }
            }

            // for (int i = 0; i < n; i ++) {
            //     System.out.print(ct[i] + " ");
            // }

            return (int)ct[n - 1];
        }
    }
    ```

=== "迪杰斯特拉 + 拓扑序 + 动态规划"

    上述造成时间复杂度和空间复杂度贼高的一大原因：允许最短路长度相同的路径也丢进去，造成了大量的冗余。

    如果我们只记录能造成当前节点 $u$ 的父节点呢？将这些父节点丢入一个集合中。

    我们用迪杰斯特拉遍历完所有顶点的最短路后，用这些父节点构建一副新的图，该图的任意一条以 $0$ 为起点 $n - 1$ 为终点的路径，都是最短路。

    我们要统计从 $0$ 开始到 $n - 1$ 总共有多少条路径。

    怎么统计？

    如果我们可以将该图转换成拓扑序，只有前面的点指向后面的点，那么从拓扑序的前面开始往后面遍历，每遍历一个点 $u$ 都将 $u$ 的所有子节点的可达路径数递增 $1$，就可以不断的推导出所有节点到 $0$ 号节点的可达路径数。

    时间复杂度：$n ^ 2、mlog(m)$ 取最大.

    ```java
    class Solution {
        long inf = ~(1L << 63);
        int N = 100000;
        long mod = 1000000007L;

        class Node {
            int a; long b; int c;
            Node(int x, long y, int z) {
                a = x; b = y; c = z;
            }
        }

        int n;
        int[][] es;
        int[] g, e, ne;
        long[] d;
        long[] md;
        long[] on;
        int cur;

        int[] topu;
        int tt;
        int[] du;

        long[] dp;

        long res;

        List<List<Integer>> ls = new ArrayList<List<Integer>>();
        List<List<Integer>> son = new ArrayList<List<Integer>>();

        void add(int a, int b, int c) {
            e[cur] = b; d[cur] = c;
            ne[cur] = g[a]; g[a] = cur;
            cur ++;
        }

        java.util.Queue<Node> qu;

        public int countPaths(int ns, int[][] roads) {
            n = ns;
            es = roads;

            N = n * n;

            g = new int[N];
            e = new int[N];
            ne = new int[N];
            d = new long[N];
            md = new long[N];
            on = new long[N];

            topu = new int[N];
            du = new int[N];
            dp = new long[N];

            for (int i = 0; i < n; i ++) {
                ls.add(new ArrayList<Integer>());
                son.add(new ArrayList<Integer>());
            }

            for (int i = 0; i < N; i ++) {
                g[i] = -1; e[i] = -1; ne[i] = -1;
                md[i] = inf;
            }

            for (int[] nd : es) {
                add(nd[0], nd[1], nd[2]);
                add(nd[1], nd[0], nd[2]);
            }

            qu = new PriorityQueue<>(new Comparator<Node>(){
                public int compare(Node x, Node y) {
                    if (x.b < y.b) return -1;
                    if (x.b > y.b) return 1;
                    return 0;
                }
            });

            qu.offer(new Node(0, 0, -1));

            while (qu.size() > 0) {
                Node aa = qu.poll();

                if (on[aa.a] > 0 && md[aa.a] < aa.b) continue;

                ls.get(aa.a).add(aa.c);

                if (md[aa.a] == aa.b) { 
                    continue;
                }
                on[aa.a] = 1;

                md[aa.a] = aa.b;

                for (int i = g[aa.a]; i > -1; i = ne[i]) {
                    qu.offer(new Node(e[i], aa.b + d[i], aa.a));
                }
            }

            for (int i = 0; i < n; i ++) {
                for (int j = 0; j < ls.get(i).size(); j ++) {
                    int u = i;
                    int v = ls.get(i).get(j);
                    if (v > -1) du[u] ++;
                    if (v > -1) son.get(v).add(u);
                }
            }

            java.util.Queue<Integer> qe = new PriorityQueue<>();

            for (int i = 0; i < n; i ++) {
                if (du[i] == 0) {
                    qe.offer(i);
                }
            }

            while (qe.size() > 0) {
                int aa = qe.poll();
                if (aa == -1) continue;
                topu[tt ++] = aa;

                son.get(aa).forEach((ee) -> {
                    if (ee == -1) return;
                    du[ee] --;
                    if (du[ee] == 0) qe.offer(ee);
                });
            }

            dp[0] = 1;

            // for (int i = 0; i < tt; i ++) {
            //     final int fa = i;

            //     son.get(topu[i]).forEach((ee) -> {
            //         dp[ee] += dp[topu[fa]];
            //         dp[ee] %= mod;
            //     });
            // }

            for (int i = 0; i < tt; i ++) {
                for (int ee : son.get(topu[i])) {
                    dp[ee] += dp[topu[i]];
                    dp[ee] %= mod;
                }
            }

            return (int)dp[n - 1];
        }
    }
    ```
