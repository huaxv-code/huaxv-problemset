# [215. 数组中的第K个最大元素](https://leetcode.cn/problems/kth-largest-element-in-an-array/description/?envType=study-plan-v2&envId=leetcode-75)

=== "计数排序"

    ```java
    class Solution {
        int N = 20001;
        int M = 10000;
        int[] a = new int[N];

        public int findKthLargest(int[] nums, int k) {
            int n = nums.length;

            for (int x : nums) a[M + x] ++;

            for (int i = N - 1; i >= 0; i --)
            {
                k -= a[i];

                if (k <= 0) return i - M;
            }

            return -1;
        }
    }
    ```
