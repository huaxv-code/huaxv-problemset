# 数据类型

=== "sql"

    ```sql
    tinyint     -- 1 字节整数

    smallint    -- 2 字节整数

    mediumint   -- 3 字节整数

    int         -- 4 字节整数

    bigint      -- 8 字节整数

    float       -- 4 字节小数

    double      -- 8 字节小数

    decimal(a, b)  -- 定义长度为 a 的，并且小数位数长度为 b 的小数

    decimal(4, 2)  -- 表示的范围：-99.99 ~ 99.99
    ```
