# 创建表-指定主键

=== python

    **方式一**

    ```py
    run = '''
        create table aa_t
        (
            id varchar(20) primary key
        )
    '''

    crs.execute(run)
    db.commit()
    ```

    **方式二**

    ```py
    run = '''
        create table aa_t 
        (
            id varchar(20),
            primary key(id)
        )
    '''

    crs.execute(run)
    db.commit()
    ```
