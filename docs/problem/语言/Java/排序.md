## 自定义结构体的排序

自定义结构体排序：

---

方法一：

实现接口：Comparable

```java
class Node implements Comparable<Node>{
    int a, b, c, d, id;
    public Node() {
        a = b = c = d = id = 0;
    }
    public Node(int id, int a, int b, int c) {
        this.id = id;
        this.a = a;
        this.b = b;
        this.c = c;
        init();
    }
    public void init() {
        d = a + b + c;
    }
    public int compareTo(Node tp) {
        if (d > tp.d) return -1;
        if (d < tp.d) return 1;
        if (a > tp.a) return -1;
        if (a < tp.a) return 1;
        if (id < tp.id) return -1;
        if (id > tp.id) return 1;
        return 0;
    }
}
```

方法二：

**首推**

统一实现 `Comparator<Node>` 接口内的 `compare` 方法 ：

```java
class Node {
    int id, a, b, c, d;
    public void init() {
        d = a + b + c;
    }
    public Node (int id, int a, int b, int c) {
        this.id = id; this.a = a;
        this.b = b; this.c = c;
        init();
    }
    public Node () {
        // 使用其他构造函数构造
        this(0, 0, 0, 0);
    }
}

class Cmp implements Comparator<Node> {
    public int compare(Node a, Node b) {
        if (a.d != b.d) return b.d - a.d;
        if (a.a != b.a) return b.a - a.a;
        if (a.id != b.id) return a.id - b.id;
        return 0;
    }
}

Arrays.sort(as, 1, n + 1, new Cmp());
```

## 升序

方法一：

```java
Arrays.sort(as, 1, n + 1, new Comparator<Integer> () {
    public int compare(Integer a, Integer b) {
        return a - b;
    }
});
```

---

方法二：

```java
class Cmp implements Comparator<Integer> {

    public int compare(Integer a, Integer b) {
        if (a > b) return 1;
        if (a < b) return -1;
        return 0;
    }
}

Arrays.sort(as, 1, n + 1, new Cmp());
```
