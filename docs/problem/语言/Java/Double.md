# Double

## 构造

```java
Double a = new Double(3.1415);

Double b = new Double("3.1415");

Double c = Double.valueOf("3.1415");

Double d = 3.1415;
```

---

## 类型转换

```java
Double a = 3.1415;

byte b = a.byteValue();

int c = a.intValue();

long d = a.longValue();

float e = a.floatValue();

Double f = a.doubleValue();

short g = a.shortValue();

String h = a.toString();
```

## Double.parseDouble

```java
double a = Double.parseDouble("3.1415");
```
