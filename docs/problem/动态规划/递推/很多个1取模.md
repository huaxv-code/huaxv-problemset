# 很多个 1 取模

??? question "问题描述"

    假设一个正整数 $N$ 在十进制上由 $n$ 个 $1$ 组成，求 $N$ 对 $M$ 取模的结果；

    **数据范围**

    $0 \leq N \leq 10 ^ 6$

    $1 \leq M \leq 10 ^ 6$

    **输入输出描述**

    输入只有一行，该行只有两个整数分别代表 $N、M$.

    输出只有一行，代表 $N \bmod M$ 的值.

    **示例一**

    输入：

    ```
    67 72
    ```

    输出：

    ```
    31
    ```

## 取模性质

取模运算对 *加、减、乘、指数* 运算毫无影响。

我们在构造出 $N$ 的过程中就不断地取模。

时间复杂度：$O(n)$

??? success "代码参考"

    ```c++
    LL n, m, res;

    void solve(void)
    {
        cin >> n >> m;
        for (LL i = 1; i <= n; i ++)
        {
            res *= 10; res += 1; res %= m;
        }
        cout << res << endl;
    }
    ```
