# 连续数组

查找前面是否可以丢弃掉多余的 0 或者 1.

??? success "代码参考"

    ```java
    class Solution {
        int[] sm;
        Map<Integer, Integer> m1, m0;
        int n;
        int res = 0;

        public int findMaxLength(int[] nums) {
            n = nums.length;
            sm = new int[n + 1];
            m0 = new HashMap<>();
            m1 = new HashMap<>();

            if (n == 1) return 0;

            for (int i = 1; i <= n; i ++) {
                sm[i] = nums[i - 1] + sm[i - 1];
            }

            for (int i = 1; i <= n; i ++) {
                int o = sm[i];
                int z = i - sm[i];

                if (o == z) {
                    res = Math.max(res, i);
                    continue;
                }

                if (o > z) {
                    if (m1.containsKey(o - z)) {
                        res = Math.max(res, i - m1.get(o - z));
                    } else m1.put(o - z, i);
                } else {
                    if (m0.containsKey(z - o)) {
                        res = Math.max(res, i - m0.get(z - o));
                    } else m0.put(z - o, i);
                }
            }

            return res;
        }
    }
    ```
