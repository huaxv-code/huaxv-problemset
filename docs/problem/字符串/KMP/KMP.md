# KMP

??? note "KMP 1 作为起点"

    ```c++
    LL i = 0, j = 1;
    nx[1] = 0;
    // 定义 s[0] 与任何元素都相同
    // 定义 nx[i] 为第 i 位置不匹配时所需要跳跃的下标，
    // 同时也是 1 ~ i - 1 的最大前后缀长度 + 1
    while (j <= n)
    {
        if (i == 0 || s[i] == s[j])
        {
            nx[++ j] = ++ i;
        }
        else i = nx[i];
    }
    ```
