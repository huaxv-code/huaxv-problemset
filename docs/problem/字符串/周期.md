=== "KMP"

    **推导 KMP**

    数值 $kmp[i]$ 代表子串 $1 \sim i$ 中间最大相同前后缀长度。

    例如：

    ![](img/kmp.png)

    代码上怎么实现，自己推导吧，我是记不住模板的！

    ```java
    kmp[1] = 0;

    for (int i = 1, j = 2; j <= n;) {
        if (s.charAt(i) == s.charAt(j)) {
            kmp[j] = i; i ++; j ++;
        } else {
            if (i == 1) { // 都到 1 了还不相等，就说明没有最大相同前后缀
                kmp[j] = 0; j ++; continue;
            }
            i = kmp[i - 1] + 1;
        }
    }
    ```
